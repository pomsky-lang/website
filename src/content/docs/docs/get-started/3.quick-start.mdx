## Playground

Get started instantly with the [online playground](https://playground.pomsky-lang.org/). It has a live preview, search and replace, and allows you to share your code!

## CLI

The CLI allows you to compile Pomsky expressions in the command line. Install it via

<Tabs>
  <TabItem label='shell'>

    ```sh
    curl --proto '=https' --tlsv1.2 -LsSf https://github.com/pomsky-lang/pomsky/releases/latest/download/pomsky-bin-installer.sh | sh
    ```

  </TabItem>
  <TabItem label='powershell'>
  
    ```sh
    powershell -ExecutionPolicy Bypass -c "irm https://github.com/pomsky-lang/pomsky/releases/latest/download/pomsky-bin-installer.ps1 | iex"
    ```

  </TabItem>
  <TabItem label='NPM'>

    ```sh
    npm install -g @pomsky-lang/cli
    ```

  </TabItem>
  <TabItem label='AUR'>

    ```sh
    yay -S pomsky-bin
    ```

  </TabItem>
</Tabs>

Or download it from the [releases page](https://github.com/pomsky-lang/pomsky/releases).

<details>
<summary>Install from source</summary>

This requires that a recent Rust toolchain is installed. Instructions for how to install Rust can be
found [here](https://www.rust-lang.org/tools/install).

Install the CLI from source with

```sh
cargo install pomsky-bin
```

</details>

<details>
<summary>How to use the CLI</summary>

To find out how to use the CLI, run

```sh
pomsky --help
```

</details>

## VS Code Extension

To use Pomsky in Visual Studio Code, install [the extension](https://marketplace.visualstudio.com/items?itemName=pomsky-lang.pomsky-vscode) and the **CLI** (see above).

## JavaScript plugin

Pomsky can be used with the npm module [`@pomsky-lang/unplugin`](https://www.npmjs.com/package/@pomsky-lang/unplugin). This is a compiler plugin, usable with Vite / Rollup / Webpack / ESBuild / ESM.

If you're using Vite, add it to your config like this:

```js
import { defineConfig } from 'vite'
import pomsky from '@pomsky-lang/unplugin'

export default defineConfig({
  plugins: [pomsky()],
})
```

Then you can import `*.pom` files from JavaScript/TypeScript:

```js
import myRegex from './myRegex.pom'

myRegex().test('does this string match?')
```

or declare Pomsky expressions with the built-in `pomsky$` macro:

```js
const myRegex = pomsky$(`% [word]{5} %`)

myRegex().test('does this string match?')
```

## Rust macro

If you want to write a Pomsky expression directly in your Rust source code, the
[pomsky-macro](https://crates.io/crates/pomsky-macro) got you covered. Run this command:

```sh
cargo add pomsky-macro
```

Then you can import and use it with

```rs
use pomsky_macro::pomsky;

const MY_REGEX: &str = pomsky!(["great!"] | "great!");
```

Documentation can be [found here](https://docs.rs/pomsky-macro/latest/pomsky_macro/).

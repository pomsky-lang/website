Example: Numbers

This regular expression matches rational numbers in decimal notation with optional separating commas:

[-+]??\b(?:0|[1-9](?:,??[0-9])*)(?:\.[0-9]+)?\b

Equivalent pomsky expression:

['-+']?
%
('0' | ['1'-'9'] (','? ['0'-'9'])*)
('.' ['0'-'9']+)?
%
